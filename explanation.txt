Your program implements the **Round Robin (RR) CPU Scheduling Algorithm**, which is a preemptive scheduling algorithm where each process gets a fixed time slice (Time Quantum).

I’ll explain your code step-by-step, then show a **simpler version** of Round Robin at the end.

---

# ✅ What Round Robin Does

* Each process gets CPU for **time quantum (tq)**.
* If it finishes → remove it.
* If not → put it back in the ready queue.
* Continue until all processes finish.

## 5️⃣ Main Scheduling Loop

---

### 🟢 Case 1: Queue is Empty

```c
if(front == rear)
```

CPU is idle:

* Increase time
* Check for new arrivals

---

### 🟢 Case 2: Execute Process

```c
int i = queue[front++];
```

Take next process from queue.

---

### 🟢 Execute for Time Quantum

If remaining time > time quantum:

```c
time += tq;
p[i].rt -= tq;
```

Else:

```c
time += p[i].rt;
p[i].rt = 0;
p[i].ct = time;
completed++;
```

---

### 🟢 Add Newly Arrived Processes

After time increases:

```c
if(p[j].at <= time && !visited[j])
```

Add them to queue.

---

### 🟢 If Process Not Finished

```c
if(p[i].rt > 0)
    queue[rear++] = i;
```

Put back into queue.

That’s the core Round Robin logic.

# 🔹 SIMPLER Round Robin Code (Basic Version)

If you want something easier (without arrival times, file writing, Gantt chart, visited array), here is a simplified version:

```c
#include <stdio.h>

int main() {
    int n, tq;
    printf("Enter number of processes: ");
    scanf("%d", &n);

    int bt[n], rt[n];
    int wt[n], tat[n];

    for(int i = 0; i < n; i++) {
        printf("Burst Time of P%d: ", i+1);
        scanf("%d", &bt[i]);
        rt[i] = bt[i];   // remaining time
    }

    printf("Enter Time Quantum: ");
    scanf("%d", &tq);

    int time = 0;
    int done;

    do {
        done = 1;
        for(int i = 0; i < n; i++) {
            if(rt[i] > 0) {
                done = 0;

                if(rt[i] > tq) {
                    time += tq;
                    rt[i] -= tq;
                } else {
                    time += rt[i];
                    wt[i] = time - bt[i];
                    rt[i] = 0;
                }
            }
        }
    } while(!done);

    float total_wt = 0, total_tat = 0;

    for(int i = 0; i < n; i++) {
        tat[i] = bt[i] + wt[i];
        total_wt += wt[i];
        total_tat += tat[i];

        printf("P%d  WT=%d  TAT=%d\n", i+1, wt[i], tat[i]);
    }

    printf("Average WT = %.2f\n", total_wt/n);
    printf("Average TAT = %.2f\n", total_tat/n);

    return 0;
}
```
